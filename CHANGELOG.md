## [5.2.0] - 2026-06-30
### رفع شده

## [5.1.0] - 2026-06-30
### رفع شده
- همگام‌سازی خودکار سایت‌مپ: وقتی آدرس سایت‌مپ ذخیره بشه، خودکار fetch و import انجام میشه
- ساخت خودکار urls-map.json بعد از هر ایمپورت موفق (بدون نیاز به کلیک دستی)
- اولین بار که ماژول با سایت‌مپ تنظیم‌شده لود بشه، خودکار sync شروع میشه

### اضافه شده
- متد `handle_auto_import()` برای اجرای خودکار fetch → import → regeneration از طریق WP-Cron
- فایل `.gitattributes` برای حذف پوشه `.git` از ZIP توزیع

## [5.0.0] - 2025-07-14
### اضافه شده
- سیستم ارتباطی جدید (Communication_Interface) بین کامپوننت‌ها
- نسخه‌گذاری قرارداد (Contract Versioning) در URLs_Map
- خواندن و نمایش وضعیت تأیید دریافت (Acknowledgment) در پنل ادمین
- تشخیص نوع صفحه از URL prefix هنگام ایمپورت سایت‌مپ
- عدم ایمپورت مجدد صفحات حذف‌شده توسط اپراتور
- فیلد content_hash (SHA-256) برای تشخیص تغییرات محتوا
- تنظیم shared_data_dir از پنل ادمین
